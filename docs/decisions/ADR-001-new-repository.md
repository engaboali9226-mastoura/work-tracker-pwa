# ADR-001

## Title

Create New Repository For V2

## Status

Approved

## Context

Current V1 is stable and running.

A full refactor is required to support:

- Clean Architecture
- PWA
- Android Packaging
- Future Scaling

## Decision

Create a separate repository for V2.

## Consequences

### Positive

- Protect V1
- Allow aggressive refactoring
- Safer experimentation

### Negative

- Temporary duplication
- Two repositories to maintain during migration
